What is Achieve?

In short Achieve is a scheduling web application that allows users to store client, staff, and team info in a database. Using that information Achieve can generate a daily schedule. This schedule is downloadable as a "CSV" file. Which can be opened in Microsoft Excel or a simular spreadsheet program.

Features

  • - Securely* add, edit, and remove client and staff info
  • - Login Functionality
  • - Generate a daily schedule for clients and staff (In Progress)
  • - Generate downloadable CSV files for the daily schedule
  • - View selected client's or staff's teams, schedule, or info
  • - View entire schedule online
  • *full security still needs to be verified
