What is Achieve?

Description:

In short Achieve is a scheduling web application that allows users to store client, staff, and team info in a database. Using that information Achieve can generate a daily schedule for the entire database of staff and clients. This schedule is downloadable as a "CSV" file. Which can be opened in MS Excel or a simular spreadsheet program.

Features:

Current Features:

  • Securely add, edit, and remove client and staff info from database

Future Features:

  • Generate a daily schedule for clients and staff
  • Generate downloadable CSV files for the daily schedule
  • View selected client's or staff's teams, schedule, or info
  • View entire schedule online
  • Generate a weekly schedule
  • Save, download, and view shedules generated in the past
  • Mobile friendly
